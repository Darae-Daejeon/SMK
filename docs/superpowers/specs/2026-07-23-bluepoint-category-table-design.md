# Bluepoint category table

## Goal

Add one technology-category table between the Bluepoint item-list heading and the existing SMK technology list.

## Layout

- Use a two-column table: `구분` and `세부 분야`.
- Render five category rows.
- Display each category's five subfields as five separate lines in the second cell.
- Retain the existing Bluepoint table's border, header, spacing, and horizontal-overflow behavior.
- Do not alter the existing nine SMK list rows or their links.
